# **Quacker App**

**Technologies used:**

- Firebase
    - For Authentication and as a database.
- React / Next JS
    - Helps in dynamic rendering
- Tailwind CSS
    - Helps in responsive design


---
## **Features:**
- Authentication using Google Auth
- Live feed update
- Can view posts without signing in
- Comment on posts
- Edit and delete your own posts
